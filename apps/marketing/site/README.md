# Marketing Site (Static)

This is the marketing-only website.

## Run Locally

Use any static file server, for example:

```bash
cd apps/marketing/site
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Scope

- Landing and product messaging
- Pricing/privacy/terms pages
- Links out to app and repository

No runtime chat/session/account logic belongs in this site.
