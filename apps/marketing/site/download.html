Get Krusty

One binary gives you the TUI, web server, and editor integration. Free, no account required.

Build from source

git clone https://github.com/honeycomb-Technologies/Krusty.git
cd Krusty
cargo build --release -p krusty

Binary output at target/release/krusty. Requires Rust toolchain.

Desktop app

cd apps/desktop/shell
bun install
bun run build

Builds native .deb, .rpm, and .AppImage packages via Tauri.

Quick start

# Terminal mode
krusty

# Web server + PWA
krusty serve

# Editor integration
krusty acp

First run prompts for provider selection and API key.

Supported providers

Bring your own API key from any supported provider:

  • MiniMax — Default provider
  • OpenAI — API key or OAuth
  • Z.AI — GLM-5 models
  • OpenRouter — 100+ models from multiple providers
